So what exactly is Firebase?

Firebase is a suite of products by Google designed to help developers focus on created user experiences and less on backend services

Just about anything you can think of they offer

  • Secure Authentication
  • Free* Web Hosting
  • NoSQL Databasing
  • Serverless Cloud Functions
  • Mobile Analytics
  • Machine Learning
  • Cloud Messaging
  • ...and even more for mobile developers!

But like, who is actually using it?

Lots of people!




Now that we got that out of the way....

Let's break it all down!

Main Features

  • Supported in any Platform
  • Multiple Different Login Providers
    • Traditional Email/Password
    • Anonymous Login
    • Google Account
    • Facebook
    • Twitter
    • Github
    • Phone Based
    • Custom Self-Hosted Authentication System

Main Features (cont.)

  • Prebuild Login UI Library
  • Custom UI Support
  • Security Backed by Google
  • Easy to drop into exist project

Pricing

Main Features

  • Backed by Fast CDN Worldwide
  • Provides free SSL Certificates
  • Simple Deployment with Rollback Features

But.... it doesn't support everything

Sites running on services such as PHP are not supported
